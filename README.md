# Simulador de Prepago Hipotecario

Una herramienta interactiva para simular escenarios de prepago en créditos hipotecarios, permitiendo evaluar el impacto financiero de diferentes estrategias de pago anticipado.

## Características

- 💰 **Cálculo de múltiples escenarios** - Simula prepagos únicos, mensuales, semestrales y anuales
- 📊 **Visualización de datos** - Gráficos comparativos para evaluar impacto
- 🎯 **Análisis de costos** - Incluye costos de prepago y seguros
- ⚙️ **Parámetros flexibles** - Ajusta tasa, plazo, monto y frecuencia
- 📈 **Reportes detallados** - Simulaciones completas con proyecciones

## Uso

1. Abre `index.html` en tu navegador
2. Ingresa los datos de tu crédito (monto, tasa, plazo)
3. Define tu estrategia de prepago
4. Visualiza los resultados en los gráficos

## Tecnología

- React 18
- Recharts para gráficos
- CSS moderno con variables
- JavaScript puro para cálculos financieros

## Archivos

- `App.jsx` - Componente principal de la aplicación
- `App.css` - Estilos y temas
- `index.html` - Página HTML
- `strip_recharts.py` - Script utilitario Python
